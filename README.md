## Code Overview

This repository contains a Python-based analytical demonstration for the
Adaptive & Privacy-Preserving Aadhaar Authentication (APPA) framework.

The code performs:
- Age-wise biometric update analysis
- Region-wise biometric update analysis
- Conceptual Biometric Reliability Score (BRS) computation
- Adaptive authentication decision logic
- Basic data visualisation

## Dataset Note
Original dataset sourced from the UIDAI Open Data Portal.
Only sample and anonymized data is used for demonstration.

## How to Run
pip install -r requirements.txt
python analysis_appa.py
